# Star Wars Registry

## Running the application

### Prerequisites

- NodeJS v20.10.0
- An Editor, preferred VSCode
- Windows 11 / Mac / Linux
- Git

### Setup

```bash
- git clone sdfsdff
- cd star-wars
- npm install
- npm run build
- npm run dev
```

Open ```http://localhost:4000/home``` on your browser.
